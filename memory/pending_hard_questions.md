# Pending Hard Questions — 待強模型清單

> 用途：`opus` ＋ 第二意見仍拿不下的判斷題停在這裡，等使用者帶去更強的模型或親自裁決。
> 進入條件：DISPATCH_POLICY §5 天花板（已升到 opus、已做第二意見、仍無把握）。
> 寫入後必須**告訴使用者**（這不是靜默擱置區）。解決後把條目移到檔尾「已解決」。

## 格式

```markdown
### Q-YYYYMMDD-NN ｜ 一句話問題
- date: YYYY-MM-DD ｜ task: Task Card ID
- 問題全文: （含所有必要脈絡，讓沒看過本 repo 的模型也能作答）
- 已嘗試: opus 的答案摘要＋第二意見的答案摘要＋兩者分歧點
- 卡住的原因: 缺什麼（判準？資訊？還是純品味取捨？）
- 影響: 不解決會擋住什麼
```
