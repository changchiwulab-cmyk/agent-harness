## 2. 四層成本分析

### 損益平衡計算（以多功能蔬果切碎器為例）

| 指標 | 數值 |
|------|------|
| 售價（USD） | 29 |
| 產品成本 | 5 |
| 國際運費 | 3 |
| 平台費（TikTok 6% + 金流） | ~2.6 |
| 退貨預備金（8%） | ~2.3 |
| **每單毛利** | **16.1（55.5%）** |
| 廣告每單成本（ROAS 3.5x） | ~8.3 |
| **每單淨利** | **~7.8（26.9%）** |
| **損益平衡訂單數（覆蓋月費 USD 690）** | **~89 單/月** ← 三品類最高 |

### ROI 估算

- **啟動成本回收**：89 單/月才能覆蓋月費；需要 TikTok 流量夠強
- **6 個月 ROI 目標**：月 200 單 × USD 7.8 = USD 1,560 淨利/月
- **回本期**：約 **4-5 個月**（需爆款帳號加持）
- **關鍵差異**：廚房品類「要麼爆，要麼死」，損益平衡最高，但爆發時利潤規模也最大

## 3. 切入選項評估

### 選項 A：TikTok 爆款廚房小工具（推薦主力）
- **產品**：多功能蔬果切碎器、電動打蛋器
- **通路**：TikTok Shop 主力，搭配 Pinterest 長尾
- **優勢**：衝動購買力最強品類，有機流量上限高
- **風險**：爆款週期短，3 個月後需換品

### 選項 B：環保廚具（歐洲市場）
- **產品**：竹製砧板、矽膠保鮮袋
- **通路**：Etsy + 歐洲 Shopify 站
- **優勢**：Green Claims Directive 2026 帶動需求，溢價空間高
- **風險**：歐洲物流成本高（運費 USD 12-18/件）

### 選項 C：不做（對照）
- 損益平衡門檻最高（89單），爆款依賴性強，非穩健風格一人公司的首選

## 4. 高風險假設

1. **假設**：TikTok 廚房帳號可在 30 天內達到 ROAS 3.5x  
   **驗證方式**：USD 300 測試廣告，目標 CPA ≤ USD 9；未達則暫停

2. **假設**：主力 SKU 爆款週期 ≥ 90 天（足以回本）  
   **驗證方式**：30 天追蹤訂單趨勢，若月成長率 <10% 則準備換品

3. **假設**：蔬果切碎器無 FDA 食品接觸安全問題  
   **驗證方式**：確認供應商提供 LFGB/FDA 認證文件
