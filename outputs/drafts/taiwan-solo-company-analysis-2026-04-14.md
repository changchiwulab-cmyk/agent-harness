## 摘要

台灣「一人公司」在法律上沒有單一定義，而是分散在獨資行號、有限公司（一人股東）、閉鎖性股份有限公司等多種形式中。**廣義的一人事業體（含獨資行號）總數估計超過 120 萬家，佔全台企業總數的 70% 以上。** 以服務業、零售業、餐飲業為最大三類，其中數位創業（電商、自媒體、顧問）在 2020 年後快速成長。

## 二、「一人公司」的定義與法律形式

台灣沒有「一人公司」的官方法律類別，實務上包含三種形式：

### 形式一：獨資行號（最主要）

| 項目 | 說明 |
|------|------|
| 法律依據 | 商業登記法（非公司法） |
| 設立主管機關 | 各縣市政府建設局/商業處 |
| 法人格 | **無**（業主個人承擔無限責任） |
| 資本額要求 | 無下限（NT$0 亦可） |
| 稅務 | 以個人綜合所得稅申報 |
| 現有登記數量 | **約 93.5 萬家**（2024 年 6 月） |

獨資行號是台灣一人事業體的**主力形式**，佔廣義一人公司總數的 75% 以上。俗稱「行號」、「個人商號」、「工作室」（後者為非正式稱呼）。

### 形式二：有限公司（一人股東）

| 項目 | 說明 |
|------|------|
| 法律依據 | 公司法（2015 年修正允許一人股東） |
| 設立主管機關 | 經濟部商業署/各地商業處 |
| 法人格 | **有**（有限責任，公司資產與個人分離） |
| 資本額要求 | 無法定下限（實務 NT$1 元可設立） |
| 稅務 | 以營利事業所得稅（20%）申報 |
| 有限公司總數 | 約 40 萬家（估計一人股東佔 55–65%） |
| **一人股東有限公司估計** | **約 22–26 萬家** |

2015 年公司法修正前，有限公司至少需要 2 位股東。修正後一人即可設立，大幅降低成立門檻，吸引大量獨資行號「升級」。

### 形式三：閉鎖性股份有限公司

| 項目 | 說明 |
|------|------|
| 引入時間 | 2015 年（公司法第 356-1 條） |
| 特色 | 股份轉讓受限、彈性章程設計、適合新創 |
| 一人設立 | 允許 |
| 現有數量 | **較少**，估計 1–3 萬家（以新創為主） |

### 形式四：自營作業者（勞動統計視角）

| 項目 | 說明 |
|------|------|
| 定義 | 無雇主、獨自工作的自雇者（主計總處分類） |
| 包含形式 | 獨資行號、接案個人、農漁業個體戶 |
| 估計人數 | **約 85–95 萬人**（含農業） |
| 非農自營作業者 | 約 60–70 萬人 |

### 趨勢：數位經濟帶動新型態一人公司（2020 年後）

| 新型態 | 說明 |
|--------|------|
| **電商經營者** | Shopee、蝦皮、Shopify 個人賣家，多以獨資行號登記 |
| **自媒體 / KOL** | YouTube、Instagram、Podcast，通常以行號或個人接案形式 |
| **AI / 數位顧問** | 後疫情期間快速成長，多以有限公司或行號登記 |
| **遠距接案（Freelancer）** | 設計、工程、文案，混合行號與個人名義 |
| **代操 / 社群行銷** | 個人代操廣告、SEO，快速成長新型態 |

## 七、關鍵洞察

1. **台灣一人事業體規模龐大但分散**：廣義超過 120 萬家，但平均規模小、存活年限短（5 年存活率約 50–60%）。

2. **獨資行號是主流但有侷限**：93.5 萬家中大多數年營收低於 200 萬，難以進入政府採購市場或取得銀行信用，部分正「升級」為有限公司。

3. **2015 年公司法修正是轉折點**：允許一人股東設立有限公司後，個人創業者有了兼顧有限責任與稅務彈性的選項，一人有限公司成長明顯。

4. **數位創業加速結構轉型**：電商、顧問、AI 應用等新型態一人公司，以有限公司設立的比例高於傳統行業，平均資本額與收入也較高。

5. **一人公司的天花板是「人力」**：台灣一人事業體最大成長障礙是個人時間，AI 工具（包含本框架的使用情境）正在改變這個限制。
