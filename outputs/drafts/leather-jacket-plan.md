# 皮衣類電商顧問：一人公司 90 天行動計畫

**任務 ID**：20260411-008
**完成日期**：2026-04-11

---

## 一、服務定位

> **「我協助皮衣品牌在永續材質轉型過程中，建立可信的環保主張，讓歐盟法規成為競爭優勢而非罰款來源。」**

### 差異化主張（三點）
1. **窄而深**：只做皮衣品類的永續轉型顧問，不做通才時尚顧問
2. **法規驅動**：EU 綠色主張指令（2026）讓服務從「加分選項」變成「必選項」
3. **亞洲視角**：協助亞洲皮衣品牌（韓國、台灣）西進時的永續合規策略

---

## 二、目標客戶輪廓（ICP）

| 屬性 | 描述 |
|------|------|
| **客戶類型** | 有自有生產能力的皮衣 DTC 品牌；或傳統皮衣 OEM 廠商想轉 DTC |
| **規模** | 年營收 USD 1M–10M；員工 10–50 人 |
| **地理** | 韓國、台灣品牌進軍歐洲；或已在歐美市場的中小皮衣品牌 |
| **主要痛點** | 「收到歐洲買家詢問永續認證，不知怎麼回答」 |
| **決策者** | 品牌創辦人或電商總監（小品牌通常是同一人） |
| **不適合** | 純白牌代工廠；超大奢侈品牌（有自己的 CSR 部門） |

---

## 三、服務菜單

### 方案一：永續主張健康檢查（入門）
**價格**：USD 1,800
**交付**：7 個工作天
- 現有材質宣稱合規性審查（EU 法規對標）
- 「漂綠」風險評估報告（3 個風險點識別）
- 可信永續主張的建立路徑建議（書面報告）

### 方案二：永續轉型策略 90 天套裝（主力）
**價格**：USD 6,500（分期：50% 簽約，50% 第 60 天）
**交付**：90 天
- 永續材質選型報告（3 種替代材質評估，含成本/效益/供應商）
- EU 綠色主張合規路徑圖（12 個月行動清單）
- 品牌永續敘事（官網文案框架 + FAQ 草稿）
- 每月 1 次策略會議（共 3 次）

### 方案三：季度永續顧問留任
**價格**：USD 2,500 / 月（3 個月最短）
**適合**：已完成方案二、需要持續追蹤法規動態和材質市場的品牌
- 每月法規更新摘要
- 每季材質市場報告
- 2 次策略會議/月

---

## 四、90 天行動計畫

### 月一（Day 1–30）：建立知識基礎 + 首客探測

| 行動 | 目標 |
|------|------|
| 完成 EU 綠色主張指令深度研讀（官方文件 + 3 個解析文章） | 成為「可信的法規解讀人」 |
| 訪談 2 個 Textile Exchange 或 Leather Working Group 成員 | 建立產業連結 + 驗證顧問定位 |
| 發布 3 篇 LinkedIn 文章（EU 法規對皮衣品牌的影響） | 吸引目標客戶自然流量 |
| 向 5 個符合 ICP 的品牌發送個人化冷信 | 市場驗證 + 探測付費意願 |
| 完成方案一（健康檢查）的模板和 SOP | 可快速交付的服務就緒 |

**月一目標**：獲得 1 個付費方案一客戶

---

### 月二（Day 31–60）：執行 + 建立 SOP

| 行動 | 目標 |
|------|------|
| 執行第一個付費案例（方案一或方案二） | 真實案例 + 學習 |
| 建立永續材質資料庫（10+ 替代材質，含供應商聯絡資訊） | 方案二交付核心資產 |
| 加入 1 個皮革業 LinkedIn 群組並貢獻內容 | 擴展觸及 + 建立公信力 |
| 向第一個客戶請求轉介紹 | 啟動口碑管道 |

**月二目標**：第二個付費客戶提案；案例研究草稿完成

---

### 月三（Day 61–90）：複製 + 調整商業模式

| 行動 | 目標 |
|------|------|
| 完成第一個完整案例 + 撰寫案例研究 | 可公開的社會證明 |
| 發布案例研究於 LinkedIn + 申請 1 個業界媒體（如 Vogue Business）投稿 | 提升公信力 |
| 評估是否引入方案三（季度留任）模型 | 收入穩定化 |
| 嘗試定價調漲（方案一 → USD 2,200，方案二 → USD 7,500） | 測試市場定價彈性 |

**月三目標**：完成 2 個付費案例；評估是否招募 1 個兼職研究助理
