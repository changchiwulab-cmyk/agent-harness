# 男性長褲電商顧問：一人公司 90 天行動計畫

**任務 ID**：20260411-016
**完成日期**：2026-04-11

---

## 一、服務定位

> **「我協助男性長褲電商品牌，在 90 天內將退貨率降低 20%，同時不降低轉換率——只靠商品說明和尺寸系統優化，不需要換 ERP。」**

### 差異化主張（三點）
1. **問題超具體**：退貨率降低，不是「品牌定位」這類模糊概念
2. **不需要大投資**：不靠換系統，靠優化現有頁面和政策
3. **90 天可量化**：有明確的前後對比數據

---

## 二、目標客戶輪廓（ICP）

| 屬性 | 描述 |
|------|------|
| **客戶類型** | 男性長褲 DTC 電商品牌；或有男性長褲品類的多品類男裝品牌 |
| **規模** | 月營收 USD 50K–500K；電商退貨率 > 25% |
| **地理** | 全球，遠端服務 |
| **最痛的點** | 「我知道退貨率高，但不知道從哪裡下手改，也不想換平台」 |
| **決策者** | 電商負責人或創辦人 |
| **不適合** | 退貨率 < 15%（問題不夠痛）；純 Amazon 賣家（平台政策限制顧問空間） |

---

## 三、服務菜單

### 方案一：退貨率快速診斷（入門）
**價格**：USD 1,500
**交付**：7 個工作天
- 退貨數據分析（要求客戶提供 6 個月退貨原因分類）
- Top 5 退貨原因根本分析
- 3 個立即可執行的改善建議（不含執行）

### 方案二：90 天退貨率優化專案（主力）
**價格**：USD 5,500（分期：50% 起始，50% 第 60 天）
**交付**：90 天完整執行
- 尺寸系統重設計（建立品牌標準量測指南）
- 全系列商品頁退貨風險評估 + 優化
- 退貨政策重寫（降低退貨意願，同時維持競爭力）
- Fit Technology 工具選型報告（2 個推薦工具 + ROI 試算）
- 每月進度報告（共 3 份，含退貨率追蹤）

### 方案三：男性轉換率提升加購（附加模組）
**價格**：USD 2,000（僅可搭配方案二購買）
- 男性購買決策路徑審查（6 個觸點優化）
- 購物車遺棄率分析 + 3 個挽回策略

---

## 四、90 天行動計畫

### 月一（Day 1–30）：建立定位 + 首客

| 行動 | 目標 |
|------|------|
| 建立 LinkedIn 主頁：「男性長褲退貨率優化顧問」 | 清晰定位上線 |
| 發布 3 篇 LinkedIn 文章（男性褲電商退貨率數據 + 案例 + 解法） | 建立搜尋可見度 |
| 找 8 個符合 ICP 的品牌，發送個人化冷信（提供免費 30 分鐘退貨診斷） | 探測市場 |
| 完成方案一的審查模板（退貨診斷問卷 + 分析框架） | 服務就緒 |
| 完成 3 個免費診斷會議 | 驗證定位 + 練習銷售話術 |

**月一目標**：1 個付費方案一客戶（USD 1,500）

---

### 月二（Day 31–60）：執行 + 建立核心工具

| 行動 | 目標 |
|------|------|
| 執行方案一或方案二（第一個案例） | 真實數據 + 學習 |
| 建立「尺寸說明最佳實踐」資料庫（10 個行業優秀範例） | 方案二核心交付資產 |
| 建立 Fit Technology 工具比較表（Fit Analytics, True Fit, Sizebay, Kiwi Sizing） | 方案二工具選型模組 |
| 在 1 個男裝 FB 社群或 Slack 貢獻 3 篇有價值文章 | 拓展觸及 |

**月二目標**：第一個案例進行中；發出第 2 個方案二提案

---

### 月三（Day 61–90）：完成案例 + 調整定價

| 行動 | 目標 |
|------|------|
| 完成第一個案例 + 撰寫詳細案例研究（含退貨率前後對比） | 最強銷售工具 |
| 發布案例研究，同時主動聯繫 3 個媒體（男裝電商相關部落格/電子報） | 擴大公信力 |
| 嘗試定價調漲（方案一 → USD 1,800，方案二 → USD 6,500） | 測試定價彈性 |

**月三目標**：2 個付費客戶；收入 USD 7,000–11,500

---

| 指標 | 目標 |
