# 運動鞋電商顧問：一人公司 90 天行動計畫

**任務 ID**：20260411-012
**完成日期**：2026-04-11

---

## 一、服務定位

> **「我協助認真型球鞋轉售賣家，用數據決策選款和定價，將每月獲利提升 30% 以上。」**

### 差異化主張（三點）
1. **數據驅動**：不靠感覺選款，靠 StockX/GOAT 價格曲線和稀缺性分析做決策
2. **賣家端**：不服務品牌，只服務轉售賣家（定位清晰，避免衝突）
3. **訂閱模型**：月度選款報告 + 定價提示，讓顧問服務有可預期收入

---

## 二、目標客戶輪廓（ICP）

| 屬性 | 描述 |
|------|------|
| **客戶類型** | 進階球鞋轉售賣家（Serious Resellers），非業餘愛好者 |
| **規模** | 月成交量 10–50 雙；月 GMV USD 3,000–15,000 |
| **地理** | 東南亞（越南、新加坡、泰國）或台灣 |
| **主要痛點** | 「靠感覺買了一批結果跌價，每次都在賭」 |
| **不適合** | 月成交 1–2 雙的業餘玩家（預算不足）；已有自己數據分析系統的大賣家 |

---

## 三、服務菜單

### 方案一：球鞋市場月報（訂閱制）
**價格**：USD 149 / 月
**交付**：每月 1 份
- 當月 Top 10 潛力款分析（含價格趨勢、稀缺性評分）
- 下個月值得關注的 Drop 日曆
- 3 個過熱市場警告（哪些款式溢價已過高，不建議囤貨）

### 方案二：個人化選款策略諮詢（月度）
**價格**：USD 490 / 月（最少 3 個月）
**交付**：每月 2 次 60 分鐘視訊 + 書面選款建議
- 客製化選款建議（依客戶資本規模、風險偏好、平台選擇）
- 定價時機提示（何時出貨獲利最大）
- 多平台分配策略（哪款放 GOAT、哪款放在地平台）

### 方案三：Drop 日套利速報（一次性服務）
**價格**：USD 299 / 次
**適合**：重要 Drop 日前（如 Jordan 1 新色、Yeezy 補貨）需要快速決策的賣家
- Drop 前 48 小時：預期溢價分析 + 囤貨建議
- Drop 後 72 小時：實際價格走勢報告 + 出貨時機建議

## 四、90 天行動計畫

### 月一（Day 1–30）：建立數據基礎 + 驗證訂閱模型

| 行動 | 目標 |
|------|------|
| 每天追蹤 StockX/GOAT Top 50 款式的價格變動，建立個人數據模型 | 3 個月後形成選款直覺 + 數據資產 |
| 自己操作 3–5 筆轉售（小資本，用真實交易學習市場節奏） | 建立「我真的在做」的公信力 |
| 在 Reddit r/sneakermarket、台灣/越南球鞋 FB 社群提供 5 次免費選款意見 | 建立社群公信力 + 探測付費意願 |
| 發布免費月報 #1（試用版），在社群免費分享 | 蒐集 20+ 人的訂閱意願回饋 |

**月一目標**：確認至少 5 人願意付費訂閱（月報）

---

### 月二（Day 31–60）：啟動付費服務

| 行動 | 目標 |
|------|------|
| 正式推出方案一月報（USD 149/月），限量 30 個名額 | 首批訂閱者 10+ 人 |
| 向月報訂閱者提案方案二（個人化諮詢），轉換率目標 20% | 2 個方案二客戶 |
| 在 1 個重要 Drop 日前發布方案三（速報），測試一次性付費意願 | 首次方案三成交 |
| 建立 Notion 模板：選款決策框架（6 個維度） | 服務標準化資產 |

**月二目標**：月收入 USD 2,000–4,000

---

### 月三（Day 61–90）：擴大訂閱基礎 + 調整定價

| 行動 | 目標 |
|------|------|
| 月報訂閱者擴展至 25–30 人 | 月費收入 USD 3,725–4,470 |
| 方案二客戶累積至 5 人 | 月諮詢收入 USD 2,450 |
| 分析哪些選款建議最受歡迎，精煉月報格式 | 提升訂閱留存率 |
| 評估是否建立付費社群（Discord）作為社群版方案一 | 探索規模化可能性 |

**月三目標**：月收入 USD 6,000+；訂閱留存率 > 80%
