## 已知事實

- 使用者深度整合 Claude 生態系（Claude Pro + Claude Code CLI），為核心生產力工具
- 使用者以一人公司模式運作於台灣與越南雙市場
- 使用者具 20+ 年跨產業管理顧問背景
- agent-harness 框架已建立並運作中，使用 git 做 checkpoint
- 台灣市場以 LINE 為主要商務通訊管道
- GitHub 為程式碼與專案管理平台

## 合理推論

- 越南市場使用 Zalo 作為主要通訊管道（越南市占最高的通訊軟體）
- 管理顧問業務需要 Zoom 或同類視訊工具進行遠端諮詢
- Google Workspace 作為行政基礎設施（郵件、日曆、文件）
- LinkedIn 為 B2B 顧問業獲客的主要專業社群平台

## 待驗證

- 具體使用哪個會計軟體（QuickBooks / XERO / 其他）
- 越南市場的行政合規工具需求
- 自動化工具的實際採用情況（Zapier / Make / 自訂腳本）
- 內容行銷平台的選擇（Medium vs Substack vs 自建）
- 知識管理工具的偏好（Obsidian vs Notion vs 其他）
- 是否需要 CRM 工具管理客戶關係
