# Analysis Skill — 評測範例

## 好的輸出範例

> 題目：評估「是否要進入越南市場」的 Go/No-Go 決策

```markdown
# 決策分析：是否進入越南市場

## 結論與建議
建議選擇「有條件進入（選項 B）」：在台灣營收穩定至 NT$150K/月後，以越南作為第二市場。
立即全面進入（選項 A）的資源需求超過一人公司當前可承擔範圍；不進入（選項 C）則放棄已有的
具體業務機會（現有 3 個 warm leads）。

## 選項比較

### 選項 A：立即全面進入越南市場
| 維度 | 評估 | 依據 |
|------|------|------|
| 價值 | 高 | 越南 AI 顧問市場年增 30%（2026 IDC 報告） |
| 成本 | 高 | 出差×4/年 NT$200K + 法律設立 NT$80K |
| 風險 | 高 | 語言障礙、法規不熟悉、台灣主業分心 |
| 可行性 | 中 | 已有 3 個越南 warm leads，但無在地人脈 |
| 執行難度 | 高 | 雙市場同時運作對一人公司超載 |
| 預期回報 | 12 個月後 NT$80K/月 | 基於 leads 轉換率 30% 估算 [待驗證] |
| 一人公司適配度 | 低 | 無法在雙市場同時維持服務品質 |

### 選項 B：有條件進入（台灣穩定後）
| 維度 | 評估 | 依據 |
|------|------|------|
| 價值 | 高 | 同上，機會保留 |
| 成本 | 中 | 延後投入，出差節省 NT$100K |
| 風險 | 中 | 台灣先穩，資源分配更可控 |
| 可行性 | 高 | 先建立台灣案例，越南 leads 培養中 |
| 執行難度 | 中 | 序列化執行，不同時超載 |
| 預期回報 | 18 個月後 NT$80K/月 | 延後 6 個月進入 |
| 一人公司適配度 | 高 | 符合一次一個主力市場的原則 |

### 選項 C：不進入越南
| 維度 | 評估 | 依據 |
|------|------|------|
| 價值 | 低 | 放棄成長機會 |
| 成本 | 低 | 零額外投入 |
| 風險 | 低 | 無新風險，但台灣市場競爭加劇 |
| 可行性 | 高 | 最容易執行 |
| 執行難度 | 低 | — |
| 預期回報 | 0 | — |
| 一人公司適配度 | 高 | 簡單，但錯失機會 |

## 高風險假設
- 「3 個 warm leads 會在 18 個月內轉換」：如不成立，選項 B 的預期回報接近 0，需重新評估
- 「台灣市場可在 6 個月內穩定至 NT$150K/月」：如不成立，進入越南的條件將持續延後

## 待驗證
- 越南 AI 顧問市場年增 30% 數據的具體來源與定義：查閱 2026 IDC 或 Gartner 越南報告
- 3 個 warm leads 的購買意願與預算規模：本月安排一次 discovery call

## 建議下一步
1. 本月確認 3 個 leads 的預算與時程（低成本、高資訊價值）
2. 台灣業務目標設定為 NT$150K/月，達到後啟動越南 pilot
3. 加入 1 個越南在地顧問社群（線上），建立 0 成本人脈
```

**為什麼好**：結論先行且有理由；每個選項都有七維評估；數字具體（NT$200K 不是「成本高」）；
高風險假設說明了不成立時的影響；「不做」選項有完整評估；建議可立即行動。

## 壞的輸出範例

> 同一題目的錯誤輸出

```markdown
# 越南市場分析

## 優缺點

**進入越南的優點：**
- 市場大
- 成長快
- 有機會

**進入越南的缺點：**
- 成本高
- 語言不同
- 風險大

## 建議
可以考慮進入越南市場，但需要謹慎評估。台灣業務也要繼續維持。
```

**哪裡錯**：
- 只列優缺點，沒有選項比較（至少要有「進」vs「不進」vs「延後進」）
- 完全沒有「不做」選項
- 成本/風險沒有量化（「成本高」高多少？）
- 結論模糊（「可以考慮」不是決策建議）
- 沒有高風險假設，沒有待驗證項目
- 沒有一人公司適配度評估
- 沒有建議下一步

---

## 判斷標準

| 項目 | 通過 | 不通過 |
|------|------|--------|
| 選項完整性 | 包含「不做」，至少 3 個選項 | 只比較「做 A」vs「做 B」，跳過維持現狀 |
| 六維評估 | 每個選項都填滿七維表格 | 只列優缺點，沒有結構化評估 |
| 量化程度 | 成本/回報有數字或範圍 | 只用「高/中/低」等形容詞 |
| 一人公司適配 | 明確評估資源限制與可執行性 | 忽略執行者是一人公司的限制 |
| 結論可追溯 | 結論直接引用分析中的具體依據 | 結論憑空出現，與分析脫節 |
| 高風險假設 | 列出並說明不成立的影響 | 沒有假設清單，或假設沒有影響說明 |
| 下一步行動 | 具體、可立即執行 | 「繼續研究」「多加考慮」等空洞結語 |
