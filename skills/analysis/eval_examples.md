# Analysis Skill — 評測範例

## 好的輸出範例

> 題目：評估「一人公司是否進入越南市場」的 Go/No-Go 決策

```markdown
# 決策分析：一人公司越南市場進入 Go/No-Go

## 結論與建議
**建議：有條件 Go — 以低承諾方式先驗證，6 個月內決定是否深入。**
理由：市場機會真實（顧問市場 $2.63B，14.9% CAGR），但語言門檻與信任建立週期是主要不確定性。
建議路徑：以「FDI 台企橋接」切入（語言優勢最強、競爭者最少），先做 2–3 個試點案，
再決定是否增加投入。不建議立即全力投入越南本地 SME 市場（語言門檻未解決前）。

## 選項比較

### 選項 A：全力進入越南（在地辦公室 + 本地客戶）
| 維度 | 評估 | 依據 |
|------|------|------|
| 價值 | 高（若成功） | 顧問市場 $2.63B，成長中 |
| 成本 | 高 | 簽證、辦公室、越南語學習 ~$20-30K 第一年 |
| 風險 | 高 | 語言門檻、法規不確定（AI 法剛生效）、信任週期長 |
| 可行性 | 低（短期）| 無越南語能力，無在地人脈 |
| 執行難度 | 極高 | 需要 6–12 個月準備期 |
| 預期回報 | 高但延遲 | 最快 18 個月後有穩定收入 |
| 一人公司適配度 | 低 | 一人公司的風險分散能力弱，不適合高投入賭注 |

### 選項 B：FDI 台企橋接（遠端顧問，服務台灣客戶在越南業務）
| 維度 | 評估 | 依據 |
|------|------|------|
| 價值 | 中高 | 台越 FDI 需求真實；台灣顧問天然優勢 |
| 成本 | 低 | 幾乎無額外固定成本；需偶爾出差 ~$3-5K/年 |
| 風險 | 低中 | 服務對象熟悉（台灣企業），越南端風險由客戶承擔 |
| 可行性 | 高 | 現有網絡可觸達潛在客戶 |
| 執行難度 | 中 | 需要建立越南法規知識，但不需越南語流利 |
| 預期回報 | 中，3–6 個月可見 | 台企有明確預算，決策快 |
| 一人公司適配度 | 高 | 低承諾、可逐步擴大、現金流正向 |

### 選項 C：不做（維持現狀）
| 維度 | 評估 | 依據 |
|------|------|------|
| 價值 | 零增量 | 放棄越南機會 |
| 成本 | 零 | — |
| 風險 | 低 | 但機會成本隨時間增加 |
| 可行性 | 高 | — |
| 執行難度 | 無 | — |
| 預期回報 | 零 | — |
| 一人公司適配度 | 中 | 穩定，但可能錯過窗口期 |

## 高風險假設
- 「FDI 台企橋接有足夠需求」：若台灣客戶的越南業務規模太小、不需外部顧問，選項 B 的市場比預期小。驗證方式：訪談 3–5 家台灣製造業越南廠主管。
- 「語言不是硬門檻」：若大多數越南本地決策者不使用英語，選項 B 的客戶群會縮限至外資公司。

## 待驗證
- 台灣企業越南廠對 AI 顧問的實際預算規模（目前無數據）
- 胡志明市台商圈的轉介網絡是否活躍（[待驗證]）

## 建議下一步
1. 聯繫 2–3 位在越南有業務的台灣顧問，了解市場實況（1 週）
2. 整理現有客戶名單，篩選出有越南業務的台灣企業（3 天）
3. 提出一個試點提案，測試市場接受度（2 週）
```

**為什麼好**：
- 包含「不做」選項（選項 C）
- 每個維度有具體依據，不只是「高/低」
- 推論依據清楚標示
- 結論可追溯至比較表
- 建議下一步具體可執行（有時間框架）
- 一人公司適配度是獨立評估維度

## 壞的輸出範例

> 同一題目的錯誤輸出

```markdown
# 越南市場分析

## 機會
越南市場很大，有很多機會。AI 很流行。台灣顧問有優勢。

## 挑戰
語言是個問題。文化也不同。競爭可能激烈。

## 建議
可以考慮進入越南市場，但需要謹慎評估風險。建議先做市場調查。
```

**哪裡錯**：
- 沒有列出具體選項（A/B/C）
- 沒有逐維度評估
- 「很大」「很流行」— 沒有具體數字或依據
- 沒有「不做」選項
- 結論模糊（「可以考慮」= 沒有建議）
- 「建議先做市場調查」— 這是迴避決策，不是分析結論
- 一人公司視角完全缺失

---

## 判斷標準

| 項目 | 通過 | 不通過 |
|------|------|--------|
| 選項完整性 | 包含「不做」；至少 2 個正向選項 | 只討論一個選項 |
| 評估具體性 | 成本有金額、風險有機率或情境 | 「高/低」等模糊詞 |
| 結論可追溯 | 結論明確引用比較表中的依據 | 結論憑空出現 |
| 一人公司視角 | 每個選項評估執行可行性 | 忽略資源限制 |
| 假設透明 | 高風險假設單獨列出 + 說明影響 | 假設隱藏在推論中 |
| 行動導向 | 建議下一步有時間框架和可執行動作 | 「繼續觀察」「再研究」等空話 |
