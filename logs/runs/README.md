將執行紀錄寫入此目錄，檔名格式：`[task_id]_[task-slug].yaml`。

## 缺口說明

`AUDIT_LOG.md` 記錄了 2026-04-04 的 7 筆批次任務（20260404-R01 至 20260404-O02），
但這些任務沒有對應的執行日誌檔案。

原因：這批任務執行於 Agent Harness v1.5 時期，當時尚未建立 `EXECUTION_LOG_SCHEMA.yaml`
與結構化執行日誌基礎設施。v2.0 升級（2026-04-09）後才正式引入此機制。

往後所有新任務一律產生執行日誌，不補建歷史紀錄以避免偽造執行資料。
